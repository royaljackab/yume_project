Description détaillée

Système d'affichage du HUD (Heads-Up Display).

Affiche les informations du joueur sur la partie droite de l'écran :

  • Nom du stage
  • Score
  • Vies restantes
  • Bombes restantes

Le HUD est dessiné à droite de la zone de jeu (PANEL_RIGHT)
